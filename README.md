## About The Project

The project consists of consuming sample data from the Integrated Library System API endpoint and presenting the information in a webpage
to the web user.

<!-- USAGE DETAILS -->
## Usage
```sh
  python app.py
```

<!-- API KEY -->
## Setting the API Key

Windows
```sh
  setx ILS_API_KEY "[API KEY HERE]"
```

Mac OSX/Linux
```sh
  export ILS_API_KEY="[API KEY HERE]"
```
